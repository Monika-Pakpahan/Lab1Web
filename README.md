Tag dasar dalam html yang pertama adalah tag title. Tag title digunakan untuk memberikan judul website yang ingin kita buat. Untuk membuat paragraf kita menggunakan tag html yaitu <p> dan diakhiri dengan </p>.Tag ini diletakkan diantara tag body.

![ss3](https://user-images.githubusercontent.com/59879254/112882449-31195e00-90f7-11eb-987f-6b26cbb6b017.png)
![ss4](https://user-images.githubusercontent.com/59879254/112882470-35de1200-90f7-11eb-830b-ecd3baf4a7ca.png)
![ss6](https://user-images.githubusercontent.com/59879254/112882809-a2f1a780-90f7-11eb-9b9a-14222e17e61f.png)

Adapun atribut paragraf yaitu center, left,right,justify. Dengan menggunakan atribut tersebut kita dapat mengatur letak paragraf sesuai keinginan kita. Cara menambahkannya yaitu dengan meggunakan tag <p align="">. Contohnya <p align="center">

![ss7](https://user-images.githubusercontent.com/59879254/112884005-2b247c80-90f9-11eb-8ffb-15e906f84229.png)

Berikutnya adalah menambahkan judul pada paragraf. Judul memiliki 6 level yaitu h1 amapai h6. Untuk menambahkan judul kita menggunakan tag <h1>judul</h1>. Tag ini diletakkan sebelum tag paragraf.

![ss8](https://user-images.githubusercontent.com/59879254/112884524-cd446480-90f9-11eb-9554-27f5a5ee92e3.png)

Berikutnya adalah pemformatan teks. Pemformatan ini digunakan untuk mengatur teks sesuai keinginan kita. Beberapa tag yang digunakan seperti <b>untuk membuat huruf bercetak tebal, <i> untuk membuat huruf miring, <mark> untuk menandai teks, <ins> untuk memberikan garis bawah pada teks, dan masih banyak lagi tag untuk melakukan pemformatan teks.
  
  ![ss9](https://user-images.githubusercontent.com/59879254/112885962-88b9c880-90fb-11eb-94b2-44db4e1b6792.png)

Untuk menambah tampilan website kita dapat menambahkan gambar. Untuk menambahkan gambar kita harus menyimpan file gambar ditempat kita menyimpan file dokumen html yang telah kita buat. Untuk menambahkan gambar kita menggnakan tag <img src="">. src adalah lokasi penyimpanan gambar. Setelah menambahkan gambar maka hasilnya akan sesuai dengan ukuran gambar yang kita punya. Maka untuk mengatur ukuran gambar kita bisa menggunakan tag width (lebar gambar) ataupun height (tinggi gambar). Kita juga dapat menggunakan tag title untuk memberikan judul atau nama pada gambar. 

![ss10](https://user-images.githubusercontent.com/59879254/112887046-eac6fd80-90fc-11eb-9fa9-8c7a242df7b8.png)

Berikunya adalah menambahkan link navigasi. Tag yang digunakan yaitu <nav> dan diakhiri dengan </nav>. Link digunakan untuk menghubungkan satu halaman web dengan halaman web yang lain baik internal maupun eksternal.Untuk membuat hyperlink digunakan tag <a> dan menambahkan atribut href untuk mengarahkan pada id elemen yang dituju.
